HomeTag is a **multi-user web application** designed to help people organize and remember item locations within their homes by tagging them directly on uploaded blueprints or floor plans.

The project solves a common everyday problem: *forgetting where items are stored*. HomeTag provides a visual and intuitive way to mark, store, and revisit item locations over time.

 **Live Demo:**  
https://hometag-o.onrender.com/

---

##  Features

-  **Multi-user support**  
  Each user can securely manage their own blueprints and tagged items.

-  **Blueprint Upload**  
  Users can upload images of home blueprints or floor plans.

-  **Item Tagging**  
  Items can be marked directly on the blueprint with labels or notes.

-  **Future Reference**  
  Users can revisit their blueprints at any time to view previously tagged items.

-  **Memory & Organization Aid**  
  Helps users remember item locations visually instead of relying on memory.

---

##  Use Cases

- Remember where tools, documents, or valuables are stored  
- Organize storage spaces visually  
- Assist households with long-term item tracking  
- Provide a simple reference for rarely used items
